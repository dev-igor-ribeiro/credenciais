# BoraCar – Sistema de Credenciais

Sistema de gerenciamento de credenciais de motoristas da BoraCar 🚗✅  

## 📌 Funcionalidades
- Login de usuários (PHP + MySQL)
- Painel com cards de status
- Filtro por nome e status
- Inserção, edição e exclusão de motoristas
- Exportação de dados (CSV/Excel)
- Geração automática de credenciais em PDF
- Alertas de vencimento (em até 30 dias ou vencidos)
- Backup automático diário do banco de dados

## 🛠️ Tecnologias
- **Backend:** PHP 8.1
- **Banco de dados:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **Servidor:** HostGator (Apache)

## 📂 Estrutura do projeto
- `/login/` → sistema de autenticação
- `/credenciais/` → painel de controle e gestão dos motoristas
- `/db/` → scripts de conexão e backup
- `/public_html/portfolio/` → páginas do portfólio informativo
